# Pokedex – Ionic / Angular

## Overview

This is a **Pokedex** app built with **Ionic + Angular** as part of a personal lab project. The goal is to practice Ionic components, Angular structure, and basic data handling in a mobile/web hybrid app.

> **Note**: Due to temporary internet issues, updates will be paused for about **one week**. During this time, I will be uploading **older projects** to keep my work documented.

---

## Tech Stack

* Ionic Framework 6
* Angular 15
* TypeScript
* RxJS
* Node.js / npm

---

## Setup

Install dependencies:

```bash
npm install
```

Run the project locally:

```bash
ionic serve
```

That’s it — the app will open in your browser.
