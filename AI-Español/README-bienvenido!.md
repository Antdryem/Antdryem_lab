# Bienvenidos

---

En cambios!

Este repositorio forma parte de un **curso personalizado**, diseñado para
evolucionar de manera constante.

No es un curso genérico ni cerrado.  
Es un espacio de trabajo vivo, construido desde la práctica real y el
aprendizaje continuo.

### ¿Qué es este proyecto?

Este trabajo está siendo desarrollado por mí, junto con **estudiantes de
escuelas católicas**, con el objetivo de:

- Aprender tecnología de forma crítica y responsable  
- Comprender la inteligencia artificial desde el uso práctico, no desde la moda  
- Construir conocimiento útil, aplicado y sostenible  

### Filosofía del curso

El contenido se irá:
- actualizando  
- corrigiendo  
- ampliando  

conforme avanzamos.

Aquí no hay versiones finales, solo **mejora continua**.

Bienvenidos al proceso.
