# 📲 Verificación en dos pasos
### Una capa extra de protección para las cuentas digitales

Las contraseñas ayudan a proteger las cuentas, pero no siempre son suficientes.

Por eso muchos servicios utilizan algo llamado **verificación en dos pasos**.

También se conoce como **autenticación de dos factores**.

---

# 🧠 Qué significa "dos pasos"

Normalmente iniciar sesión requiere una sola cosa:

- tu contraseña

Con verificación en dos pasos, el sistema pide **una segunda confirmación**.

Por ejemplo:

- un código enviado al teléfono  
- una notificación en una aplicación  
- un código temporal generado por un dispositivo

Esto significa que para acceder a la cuenta se necesitan **dos elementos diferentes**.

---

# ⚙ Por qué mejora la seguridad

Si alguien descubre tu contraseña, todavía necesitaría el **segundo factor de verificación**.

Eso hace mucho más difícil que un atacante pueda acceder a una cuenta.

Por eso muchas plataformas recomiendan activar esta función cuando está disponible.

---

# ⚡ Un ejemplo sencillo

Imagina que tu contraseña es la llave de tu casa.

La verificación en dos pasos sería como **tener además un sistema de alarma**.

Incluso si alguien tiene la llave, todavía necesita superar otra barrera.

---

# 🧠 Una observación importante

Muchas personas no activan esta función simplemente porque requiere un paso adicional.

Pero ese pequeño paso extra puede aumentar significativamente la protección de una cuenta.
