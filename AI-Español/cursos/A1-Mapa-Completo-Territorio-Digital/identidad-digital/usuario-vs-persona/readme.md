# 👤 Usuario vs persona
### La diferencia entre quién eres y cómo apareces en internet

Cuando usas internet normalmente lo haces a través de **cuentas**.

Por ejemplo:

- una cuenta en una red social  
- un perfil en una plataforma de videojuegos  
- un usuario en una comunidad online  

Esas cuentas representan a un **usuario** dentro de un sistema digital.

Pero un usuario no es exactamente lo mismo que una **persona**.

---

# 🧠 Qué es un usuario

Un usuario es una **identidad dentro de un sistema digital**.

Normalmente está definido por cosas como:

- un nombre de usuario  
- una contraseña  
- un perfil  
- ciertas configuraciones  

El sistema reconoce esa identidad para saber:

- quién publica contenido  
- quién interactúa  
- quién accede a ciertos servicios

---

# 🌐 Qué es una persona

Una persona es mucho más que una cuenta digital.

Tiene:

- contexto  
- emociones  
- relaciones  
- experiencias fuera de internet  

Pero los sistemas digitales no ven todo eso.

Solo ven **la información asociada al usuario dentro de la plataforma**.

---

# ⚡ Por qué esta diferencia importa

En internet una persona puede tener **muchos usuarios diferentes**.

Por ejemplo:

- un perfil público  
- una cuenta para videojuegos  
- otra para trabajo o estudio  
- otra para comunidades específicas

Cada uno representa **una parte distinta de la identidad digital**.

---

# 🧠 Una idea importante

Cuando interactuamos online, muchas veces estamos viendo **usuarios**, no personas completas.

Eso puede cambiar la forma en que las personas se comportan en internet.
