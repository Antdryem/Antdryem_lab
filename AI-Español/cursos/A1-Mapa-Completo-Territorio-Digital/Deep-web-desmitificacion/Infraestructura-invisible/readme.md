# 🏗 Infraestructura invisible
### La parte de internet que casi nunca vemos

Cuando usamos internet normalmente vemos solo una pequeña parte del sistema.

Por ejemplo:

- páginas web
- aplicaciones
- videos
- redes sociales

Pero debajo de esa capa visible existe una enorme infraestructura.

---

# 🌐 Sistemas que normalmente no vemos

Gran parte de internet está formada por sistemas que no están pensados para el público general.

Por ejemplo:

- bases de datos
- redes internas de empresas
- sistemas académicos
- servicios privados
- paneles administrativos

Estos sistemas forman una parte importante de internet, pero normalmente **no están diseñados para aparecer en buscadores**.

---

# ⚙ Por qué permanece invisible

Muchos de estos sistemas requieren:

- autenticación
- permisos específicos
- acceso restringido

Por eso no aparecen como páginas públicas en la web.

Sin embargo, siguen siendo parte de la infraestructura digital global.

---

# ⚡ Una idea importante

La parte visible de la web es solo **una pequeña fracción de todo lo que existe en internet**.

Debajo hay capas de sistemas que sostienen gran parte de la actividad digital.
