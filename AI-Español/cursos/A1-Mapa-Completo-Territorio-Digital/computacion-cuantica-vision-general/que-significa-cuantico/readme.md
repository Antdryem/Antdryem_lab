# ⚛ Qué significa “cuántico”
### Cuando la física cambia las reglas del cálculo

Las computadoras normales funcionan con electricidad y lógica binaria.

Todo se reduce a dos estados posibles:

- 0
- 1

Cada cálculo que realiza una computadora está construido con millones o miles de millones de estos pequeños interruptores.

Pero cuando los científicos estudian la materia a escalas extremadamente pequeñas, descubren algo curioso:

> el mundo microscópico no siempre sigue las reglas intuitivas que conocemos.

Ese campo de estudio se llama **física cuántica**.

---

# 🌌 El comportamiento extraño de lo muy pequeño

A escalas muy pequeñas (átomos y partículas), las cosas pueden comportarse de formas inesperadas.

Por ejemplo:

- una partícula puede estar en múltiples estados al mismo tiempo  
- ciertas propiedades solo se definen cuando se miden  
- sistemas separados pueden influirse entre sí de maneras muy extrañas  

Estas ideas pueden parecer difíciles de imaginar, pero han sido confirmadas por experimentos durante décadas.

---

# 💻 La idea detrás de la computación cuántica

La computación cuántica intenta usar estas propiedades físicas para realizar cálculos de una forma diferente.

En lugar de usar bits tradicionales (0 o 1), usa **qubits**.

Un qubit puede representar:

- 0
- 1
- o combinaciones de ambos al mismo tiempo

Esto abre nuevas posibilidades para ciertos tipos de cálculo.

---

# ⚡ Una idea importante

La computación cuántica no significa que las computadoras serán mágicamente más rápidas en todo.

Significa que **algunos problemas específicos podrían resolverse de maneras muy diferentes**.
