# ⚛ Computación cuántica (visión general)
### Una forma diferente de pensar el cálculo

Las computadoras modernas han transformado el mundo.

Desde teléfonos hasta enormes centros de datos, todas funcionan usando el mismo principio básico:

> procesar información usando bits que representan **0 o 1**.

Este modelo se conoce como **computación clásica**, y es el que ha impulsado prácticamente toda la tecnología digital que usamos hoy.

Pero algunos científicos están explorando otra forma de calcular basada en una parte muy extraña de la física:

**la mecánica cuántica**.

---

# 🌌 Cuando la física cambia las reglas

A escalas extremadamente pequeñas —como átomos y partículas— el universo no siempre se comporta como esperamos.

La física cuántica describe fenómenos donde:

- los sistemas pueden tener múltiples estados al mismo tiempo  
- algunas propiedades solo se definen cuando se miden  
- partículas separadas pueden influirse entre sí de formas inesperadas  

Estas ideas pueden parecer extrañas, pero han sido confirmadas por experimentos durante décadas.

La computación cuántica intenta aprovechar estas propiedades para realizar ciertos cálculos de forma diferente.

---

# 💻 Dos formas de procesar información

Hoy existen dos modelos principales de computación.

### Computación clásica

- usa **bits**
- cada bit es **0 o 1**
- procesa información paso a paso

### Computación cuántica

- usa **qubits**
- un qubit puede representar **múltiples estados al mismo tiempo**
- permite explorar diferentes soluciones de maneras nuevas

Esto no significa que todo será más rápido.

Significa que **algunos problemas podrían resolverse de formas completamente distintas**.

---

# 🔍 Problemas que podrían beneficiarse

Los investigadores creen que la computación cuántica podría ayudar con problemas muy complejos, como:

- simulación de moléculas y química avanzada  
- desarrollo de nuevos materiales  
- optimización de sistemas muy grandes  
- algunos problemas matemáticos difíciles  

También podría tener impacto en la **seguridad digital**, porque ciertos métodos de criptografía actuales podrían verse afectados por este nuevo tipo de cálculo.

# ⚡ Una idea importante

La computación cuántica todavía está en desarrollo.

Puede transformar algunos campos importantes, pero **no significa que todas las computadoras del futuro serán cuánticas**.

Lo más probable es que coexistamos con diferentes tipos de computación, cada una diseñada para resolver distintos problemas.
