# 🧾 Registro distribuido
### Una forma diferente de guardar información

Normalmente, cuando pensamos en guardar información, imaginamos algo centralizado.

Por ejemplo:

- un servidor
- una base de datos
- una empresa que controla los datos

Pero blockchain propone una idea diferente.

---

# 🌐 Qué significa “distribuido”

Un registro distribuido es un sistema donde:

- la información no está en un solo lugar  
- está copiada en muchos nodos (computadoras)  
- todos tienen una versión del mismo registro  

Esto significa que no hay un único punto de control.

---

# ⚙ Cómo funciona la idea

Cada vez que ocurre algo (una transacción, un cambio), se registra en el sistema.

Ese registro:

- se agrupa en bloques  
- se conecta con registros anteriores  
- se replica en múltiples nodos  

Por eso se llama **blockchain (cadena de bloques)**.

---

# ⚡ Una idea importante

En lugar de confiar en un solo servidor, el sistema funciona porque:

> muchos participantes comparten y verifican la misma información.
