# 🌐 Qué es un navegador
### La herramienta que usamos para entrar a la web

Para acceder a la mayoría de los sitios de internet usamos un **navegador**.

Un navegador es un programa que permite:

- abrir páginas web  
- interpretar código de los sitios  
- mostrar contenido como texto, imágenes y videos  

Sin un navegador, las páginas web existirían en servidores, pero **no podríamos verlas fácilmente**.

---

# 🧠 Qué hace realmente un navegador

Cuando escribes una dirección web o haces clic en un enlace, el navegador:

1. solicita la página a un servidor en internet  
2. recibe el contenido del sitio  
3. interpreta el código del sitio web  
4. lo muestra como una página visual

En otras palabras, el navegador **traduce el lenguaje técnico de la web en algo que las personas pueden usar**.

---

# ⚙ Más que un simple visor

Los navegadores también realizan otras funciones importantes:

- gestionar pestañas
- guardar historial
- almacenar contraseñas
- ejecutar aplicaciones web
- proteger contra algunos riesgos de seguridad

Hoy en día funcionan casi como **sistemas operativos pequeños dentro del internet**.

---

# ⚡ Una idea importante

Cuando usamos internet normalmente no interactuamos directamente con servidores o sistemas.

Interactuamos con ellos **a través del navegador**.
