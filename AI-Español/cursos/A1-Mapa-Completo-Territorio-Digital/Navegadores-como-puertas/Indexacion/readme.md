# 📚 Indexación
### Cómo se organiza la información de la web

La web contiene **millones de sitios y páginas**.

Si no existiera alguna forma de organizarlas, encontrar información sería extremadamente difícil.

Para resolver esto, muchos sistemas utilizan un proceso llamado **indexación**.

---

# 🧠 Qué significa indexar

Indexar significa **registrar y organizar información para poder encontrarla después**.

En el contexto de internet, la indexación consiste en:

- recorrer páginas web
- analizar su contenido
- guardar información sobre ellas en grandes bases de datos

Esto permite que después se puedan localizar rápidamente cuando alguien busca algo.

---

# ⚙ Quién hace la indexación

La indexación normalmente la realizan sistemas automáticos llamados **robots o rastreadores**.

Estos programas recorren internet siguiendo enlaces entre páginas.

Mientras lo hacen, registran información sobre los sitios que encuentran.

Con el tiempo, esto crea un enorme mapa organizado de la web.

---

# ⚡ Una idea importante

No todo el contenido de internet está indexado.

Algunas páginas no aparecen en los sistemas de búsqueda.

Eso significa que **una parte del internet existe, pero no es fácilmente visible** para quienes buscan información.
