# 🔎 Motores de búsqueda
### Las herramientas que usamos para encontrar información

Dado que la web contiene enormes cantidades de información, necesitamos herramientas para encontrar lo que buscamos.

Ahí entran los **motores de búsqueda**.

Un motor de búsqueda es un sistema que permite:

- buscar información en la web
- localizar páginas relevantes
- mostrar resultados organizados

---

# 🧠 Cómo funcionan

Los motores de búsqueda utilizan tres procesos principales:

1. **Explorar la web** con programas automáticos  
2. **Indexar la información** encontrada  
3. **Ordenar resultados** cuando alguien hace una búsqueda

El sistema intenta mostrar primero las páginas que considera más relevantes.

---

# ⚙ El papel de los algoritmos

Para decidir qué resultados aparecen primero, los motores de búsqueda utilizan **algoritmos**.

Estos algoritmos analizan factores como:

- relevancia del contenido
- popularidad de una página
- estructura de enlaces entre sitios

Esto determina qué páginas aparecen en la parte superior de los resultados.

---

# ⚡ Una idea importante

Cuando buscamos algo en internet, no estamos viendo toda la web.

Estamos viendo **la parte que el motor de búsqueda decidió mostrarnos primero**.
