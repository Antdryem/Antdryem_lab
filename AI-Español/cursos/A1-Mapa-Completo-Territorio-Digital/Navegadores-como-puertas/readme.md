# 🧭 Navegadores como puertas
### Cómo accedemos realmente a la web

Cuando usamos internet normalmente pensamos que estamos explorando la web directamente.

Abrimos una página, hacemos una búsqueda o seguimos un enlace.

Pero entre nosotros y el resto de internet existe una capa importante de herramientas.

Esas herramientas funcionan como **puertas de acceso a la web**.

Entre las más importantes están:

- los navegadores
- los sistemas que organizan información
- los motores de búsqueda

---

# 🌐 Internet es demasiado grande para recorrerlo sin ayuda

La web contiene millones de sitios y páginas conectadas entre sí.

Sin herramientas para explorar esa información, encontrar algo específico sería extremadamente difícil.

Por eso existen sistemas que permiten:

- abrir páginas web
- organizar contenido
- localizar información relevante

Estas herramientas hacen posible navegar el enorme espacio digital que conocemos como internet.

---

# ⚙ Las herramientas que usamos para navegar

Cuando exploramos la web normalmente interactuamos con tres elementos clave.

Primero, **los navegadores**, que permiten abrir y visualizar páginas web.

Luego, sistemas que realizan **indexación**, organizando información de millones de páginas para que pueda encontrarse.

Finalmente, **los motores de búsqueda**, que utilizan esa información para mostrar resultados cuando buscamos algo.

Estas tres piezas trabajan juntas para hacer posible la navegación en internet.

---

# 🧩 Lo que exploraremos en este bloque

Para entender cómo accedemos a la web veremos tres conceptos fundamentales.

## 🌐 Qué es un navegador
El programa que usamos para abrir y visualizar páginas web.

## 📚 Indexación
El proceso mediante el cual sistemas automáticos registran y organizan información de la web.

## 🔎 Motores de búsqueda
Las herramientas que permiten encontrar información dentro del enorme contenido de internet.

---

# ⚡ Una idea importante

Cuando usamos internet no estamos viendo todo lo que existe en la web.

Estamos viendo solo **la parte que las herramientas de navegación nos permiten encontrar**.

Comprender cómo funcionan estas herramientas ayuda a entender mejor cómo exploramos el mundo digital.
